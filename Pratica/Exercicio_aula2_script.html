Teste para raz√£o de duas vari√¢ncias populacionais, \(\sigma_1\) e \(\sigma_2\)

\(\left\{ \begin{array}{ll} H_0: \sigma_{1}^{2} = \sigma_{2}^{2} \rightarrow \dfrac{\sigma_{1}^{2}}{\sigma_{2}^{2}} = 1 \\ H_1: \sigma_{1}^{2} \neq \sigma_{2}^{2} \rightarrow \dfrac{\sigma_{1}^{2}}{\sigma_{2}^{2}} \neq 1 \end{array} \right.\)

  • Sabemos que \(F_{calc} = \dfrac{58.92}{10.06} = 5,86\)

Supondo \(\alpha = 5\%\), os valores cr√≠ticos (na tabela F) s√£o:

\(F_{tab_1} = 1/F_{(n_{2}-1; n_{1}-1; \alpha/2)} = 1/F_{(20; 17; 0,025)} = 1/2,616 = 0,382\)

\(F_{tab_2} = F_{(n_{1}-1; n_{2}-1; \alpha/2)} = F_{(17; 20; 0,025)} \sim 1/F_{(20; 20;0, 025)} = 2,464\)

Teste para compara√ß√£o entre duas m√©dias populacionais, \(\mu_1\) e \(\mu_2\)

\(\left\{ \begin{array}{ll} H_0: \mu_1 = \mu_2 \\ H_1: \mu_1 \neq \mu_2 \end{array} \right.\)

\[S_p = \sqrt{\dfrac{(n_{A} - 1)var(x_A) + (n_{B} - 1)var(x_B)}{n_{A} + n_{B} - 2}}\] \[S_p = \sqrt{\dfrac{(18 - 1) \times 58,92 + (21 - 1) \times 10,06}{18 + 21 - 2}} = \sqrt{ \dfrac{1202,84}{37}} = \sqrt{32,51} = 5,67\] \[t_{calc} = \dfrac{\bar{x}_A -\bar{x}_B}{S_p \sqrt{ \dfrac{1}{n_{A}} + \dfrac{1}{n_{B}}}}\] \[t_{calc} = \dfrac{55,72 - 55,43}{5,67 \times \sqrt{\dfrac{1}{18} + \dfrac{1}{21}}} = \dfrac{0.29}{10,44} = 0,158\]

Supondo \(\alpha = 5\%\), os valores cr√≠ticos (na tabela t) s√£o:

\(t_{tab} = t_{(n_{1} + n_{2} - 2; \alpha)} = t_{(18 + 21 -2; 0,05)} = t_{(37; 0,05)} = 2,04\)
